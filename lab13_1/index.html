Тестирование знаний языков HTML, CSS, JS

На этой странице вы можете проверить свои знания языков HTML (Hyper Text Markup Language), CSS (Cascading Style Sheets) и JS (JavaScript).

Тест состоит из 6 вопросов - по 2 на каждый из языков. Один из этих вопросов - с выбором ответа, в другом нужно будет вписать ответ вручную.

За каждый правильный ответ дается 1 балл. Максимальное количество баллов за тест - 6.

На выполнение заданий отведено 5 минут. По истечении времени тест автоматически завершится.

Чтобы приступить к тестированию, нажмите кнопку "Начать" - с этого момента пойдет отсчет времени. Удачи!
